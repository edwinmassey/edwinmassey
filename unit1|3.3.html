3.3 Develop solutions to improve own productivity using IT in digital projects – what would you do differently next time? Team work? Use of Slack?

Solutions to productivity

As this was an individual project I felt that Slack wasn’t taken full advantage of. Within a team we could have used messages within Slack to communicate as well as share ideas or research we had found. Working within a team means we could have told one member to work on the CSS stylesheet, another on the HTML pages and having one more researching on how to implement more changes to the website. This method would have been more productive in creating a website.
