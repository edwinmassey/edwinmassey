Unit 1 Outcome 1.5

Describe how you would go about selecting and using IT systems and software – how do you select the right kind of technology for the job?

Introduction

When choosing which technology to use for a digital project it is important to select the right kind of technology for the job. In order to make sure we choose the most useful tool for the project we need to ask how do we use this tool and what benefits us from using this tool?

Deciding the right tool

So if you are creating a mobile app ensure you know how to use the tool to create the app. If you know how to use Xamarin to build mobile apps but don’t know how to use Sencha then it would be most effective to use Xamarin as you have knowledge of the tool already, saving you time instead of learning something new. Looking at benefits of which tool to use is essential as if Xamarin is easier to use but Sencha can go more in depth with coding then it may be better to learn Sencha if you have the time and are willing. Deciding what tool has the most benefits or most important benefits will allow you to choose the right kind of technology to build your project.
