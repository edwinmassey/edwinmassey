Unit 2 Outcome 1.7

1.7 describe which are the appropriate filetypes for website

Introduction

When creating our website it is important to select appropriate filetypes to use for images, audio, video and files so that our website will load quickly and work correctly.

Web images

For web images the most used file type is .JPEG or .JPG. This file type uses a low amount of storage data which means that our website can load this type of image quickly. This is important for our users to load images quickly. JPEG files can support a wide range of colours however when compressed the quality of images are lowered.

Animated images

When we want to insert a short video clip the most used file type is .GIF. This file type allows for us to include a short animation which has commonly been used on social media to insert funny clips. Keeping the clip to around 5 seconds is important otherwise it may cause our website to load for longer.

Logo images

Using a logo in our website it will be best to use .SVG as these file types are small but also allow for us to resize the image with a transparent background without much compression distortion. However SVG files may not be accessible for older browsers which in this case PNG files can be used.

Document files

For uploading document files to our website the best file type to use is .PDF. This file type will allow us to include all pages of the document onto one file, the user cannot change the text in the file either. This is also beneficial if we want the user to sign a document.

Multiple files

In our website we may want to upload multiple files and we can do this by uploading each individual file however this would take up a lot of storage space and can be hard for the user to download. Using a .ZIP file enables us to put all our files into a ZIP file where the user only needs to download one file instead of multiple and uses less storage space.

Video

To insert a video into our website we can simply use a .MP4 file to insert the video. However uploading the file to YouTube then imbedding the video will use less storage space resulting in our website loading faster. Youtube is also accessible on all devices.

Audio

The best file to use for uploading an audio file is .MP3 as this file type is accessible by all devices. This file type is the most commonly used file type across all devices for audio files.
