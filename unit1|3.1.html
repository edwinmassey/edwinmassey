Benefits

When creating the website using Trello to plan has been very time efficient. With Trello planning was made simple as I started by answering some unit questions then by creating the website, updating Trello as I did. This saved me time thinking ahead. With using Virtual studio code having one CSS stylesheet I found to be efficient as I could apply styling to all pages through the stylesheet.

Drawbacks

With this project I could have been more productive by working within a team. Working within a team would have allowed for us as a team to delegate individuals to work on a specific part of the website. Doing this would have saved a lot of time but by also working within a team there would have been more time for research and a chance to help each other to add more elements onto the website.
