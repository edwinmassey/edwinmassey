Unit 1 Outcome 2.3

2.3 Assess the strengths and weaknesses of your final project – are you happy with your website? Why? Why not? You should review it on the website itself in the interests of openness. What would you change?

Introduction

Overall I am happy with the creation of my website.

Strengths

I feel I have done well to learn how to code using HTML and CSS in a short amount of time. My website has different types of fonts, font sizes and text background boxes helping users to read. I have sorted my content into subheadings with paragraphs for clear access of information. I have included a navigation bar which is on all pages for users to reach the page they are looking for quickly. Using a background image has also kept me website looking unique to my own style.

Weaknesses

As only using CSS and HTML for a short amount of time this had led to my website being rather basic. It has necessary items such as a navigation bar but because I have a basic knowledge of CSS and HTML I couldn’t add more features to the website. Another weakness has been the time taken to get the website coded as with more time I could have asked for feedback in order to improve my website
